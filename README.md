# 📱 Ứng dụng Máy Tính Nâng Cao Flutter

## 📌 Mô tả dự án

Đây là ứng dụng máy tính nâng cao được xây dựng bằng Flutter.
Dự án mở rộng từ máy tính cơ bản thành một ứng dụng chuyên nghiệp với nhiều chế độ, chức năng khoa học và giao diện hiện đại.

Ứng dụng giúp luyện tập:

* Quản lý state nâng cao với Provider
* Xử lý biểu thức toán học
* Thiết kế UI/UX
* Lưu trữ dữ liệu
* Viết unit test

---

## 🚀 Tính năng chính

### 🔢 Chế độ máy tính

* Basic (cơ bản)
* Scientific (khoa học)
* Programmer (lập trình)

---

### 🧮 Chức năng tính toán

* Thực hiện biểu thức đúng thứ tự (PEMDAS)
* Hỗ trợ ngoặc ()
* Các hàm:

  * sin, cos, tan
  * log, ln
  * căn bậc 2 √
* Hằng số:

  * π
  * e

---

### 💾 Bộ nhớ

* M+ (cộng vào bộ nhớ)
* M- (trừ khỏi bộ nhớ)
* MR (lấy giá trị bộ nhớ)
* MC (xóa bộ nhớ)

---

### 📊 Lịch sử

* Lưu phép tính
* Xem lại lịch sử
* Xóa lịch sử

---

### 🎨 Giao diện

* Dark mode / Light mode
* Giao diện hiện đại
* Responsive
* Button bo góc

---

### ⚙️ Cài đặt

```bash

```
